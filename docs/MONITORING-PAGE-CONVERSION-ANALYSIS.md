# Анализ страницы /monitoring для повышения конверсии

**Дата анализа:** 16 декабря 2024  
**Страница:** `/monitoring`  
**Продукт:** 24/7 Care Home Monitoring Service  
**Целевая конверсия:** Подписка на мониторинг (£15-49/месяц)

---

## 📊 EXECUTIVE SUMMARY

### Текущее состояние
- **11 секций** в конверсионной воронке
- **Lead capture форма** в Hero (checklist download)
- **3 тарифа:** Essential (£19), Premium (£29), Professional (£49)
- **30-day free trial** предлагается
- **Сильные стороны:** Case study, testimonials, FAQ

### Проблемы конверсии
1. **Слабая демонстрация ценности** - нет примеров реальных отчетов
2. **Нет социального доказательства в Hero** - только в середине страницы
3. **Pricing не оптимизирован** - нет якорного эффекта
4. **Lead capture не связан с конверсией** - отдельная форма
5. **Нет urgency/scarcity** - отсутствует дедлайн
6. **Слабая демонстрация ROI** - £8,500 vs £15-49/месяц не показан визуально
